Rosie's Bakery

Keywords: html, website, coding

As a final project to a summer coding class, we were tasked with creating a website from scratch using html language.

Everyone loves desserts, but not everyone can bake them (myself included). So to make up for that, I created this website for baked goods.

This website was particularly challenging not because of the coding, but because the thumbnails were particularly distracting!

Please note that while there is a Rosie's Bakery here in Chicago, this website is unaffiliated with that organization as stated on each page.
